MAD UI Kit

Este kit contiene todo lo relacionado con el aspecto gráfico de la interfáz del "MAD".
Aquí encontrarás la biblioteca de los componentes utilizados y como agregar metodos para globalizar todas las aplicaciones relacionadas .

Paleta de colores

Las clases para cambiar d color en el MAD son:
'mrdo' sufijo de color, seguido de un guión para insertar un numero con la tonalidad: "-0"
En general quedaría: mrdo-0.

Morado
Amarillo
Verde
Rojo
<!-- COLOR MORADO -->
<div class= "mrdo-0" ></div>
<!-- COLOR VERDE -->
<div class= "vrde-0" ></div>
<!-- COLOR AMARILLO -->
<div class= "amlo-0" ></div>
<!-- COLOR ROJO -->
<div class= "rojo-0" ></div>
Reduerda que las tonalidades se ponen de acuerdo al numero que lleva al final, para consultar las tonalidades ve al menú de "Paleta de colores"

<div class= "h-vrde" ></div>

<div class= "h-amlo" ></div>

Generalmente el borde va acompañado de otros estilos, ya que el estilo de borde contiene solo el borde que se le añadira al elemento.
Por ejemplo:
<!-- EJEMPLO -->
<div class= "box h-mrdo gris-9" ></div>

En este ejemplo se puede observar que se añadio primero el estilo de "box" que es el que contiene el estilo de la caja seguido del borde "h-mrdo" y por último el fondo de color "gris-9".

Popups

Los Popovers son distintos y se pueden emplear fácilemnete con las funciones descritas en el apartado correspondiente. Para que funcionen adecuadamente hay que agregar 3 atributos a donde se quiere emplear el popup, estos atributos constarán de el valor del movimiento donde se desplegará el popup, seguido del titulo y el contenido, el sistema automaticamente reconocerá la etiqueta y le dará los valores necesarios para que funcione de acuerdo a los valores que se describen a continuación.

PopUp

Popover titulo

Sed posuere consectetur est at lobortis. Aenean eu leo quam. Pellentesque ornare sem lacinia quam venenatis vestibulum.

Ejemplos:

<!-- POPUP -->
<button data-popup= " Tipo de movimiento " data-popup-titulo= " Titulo del Popup " data-popup-cont= " Contenido del POPUP " </button>
<!-- DATA POPUP -->
data-popup=
"arrriba"
"abajo"
"izquierda"
"derecha"
"modal"
NOTA: las únicas etiquetas que funcionan con esta modalidad de popups son "DIVS","BUTTON" y "SPAN".

Checkboxes and Ratios

Los ckeckbox tienen su id en especial para que puedan ser identificados con el diseño del MAD.

<!-- CHECK -->
<input type= "checkbox" id= "BTN1" > </input>
<label for= "BTN1" > CHECK </label>
Seguido de la etiqueta input deberá tener la etiqueta label y las dos deberán tener el mismo ID.

<!-- RATIO -->
<input type= "radio" id= "BTN2" name= "BTN2" > </input>
<label for= "BTN2" ><span></span> RATIO </label>
Dentro de la etiqueta label se deberá insertar una etiqueta span con el contenido vacio.
NOTA: El único color para los checkboxes y radio buttons es morado.
